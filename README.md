- 5-day weather forecast

## Getting Started

First, install the dependency packages:

```bash
npm run install
# or
yarn
```

Second, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
